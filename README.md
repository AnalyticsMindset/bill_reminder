# Bill Reminder App

A simple, beautiful bill reminder web app built with HTML, CSS, and JavaScript. Data is stored in the browser's localStorage.

## Features

- Add recurring or one-time bills
- Track upcoming, overdue, and paid bills
- Automatic next due date calculation for monthly/yearly bills
- Search and filter
- Mobile responsive
- In-app reminders for bills due soon

## How to Deploy on Netlify

1. Go to [Netlify](https://app.netlify.com)
2. Create a new site from Git (recommended) or drag and drop this folder
3. Connect to your GitHub repo containing this folder
4. Netlify will auto-detect the static site and deploy it!

No build step needed.

## Local Development

Just open `index.html` in your browser.

## Data Persistence

All data is saved in your browser. Clearing site data will reset the app.

Enjoy staying on top of your bills! 💸
